Python Installation

1 Download Python

Visit python.org and download Python 3.11 or newer

2 Run Installer

Run the installer and IMPORTANT: Check "Add Python to PATH"

Always check "Add Python to PATH" during installation
3 Verify Installation
python --version pip --version
1 Using Homebrew (Recommended)

First install Homebrew if you don't have it:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
2 Install Python
brew install python@3.11
3 Verify Installation
python3 --version pip3 --version
Use python3 and pip3 on macOS
1 Update Package Manager
sudo apt update sudo apt upgrade -y
2 Install Python & Dependencies
sudo apt install python3 python3-pip python3-venv -y
3 Verify Installation
python3 --version pip3 --version

Virtual Environments

Virtual environments isolate project dependencies and prevent conflicts.
1 Create Virtual Environment
python -m venv data_engineer_env
2 Activate Virtual Environment

Windows:

data_engineer_env\Scripts\activate

macOS/Linux:

source data_engineer_env/bin/activate
3 Install Packages

Your prompt should show (data_engineer_env) prefix:

(data_engineer_env) $ pip install pandas numpy sqlalchemy
4 Deactivate Virtual Environment
deactivate

Essential Python Packages

1 Install Data Science Stack
pip install pandas numpy scipy scikit-learn matplotlib seaborn jupyter
  • pandas - Data manipulation
  • numpy - Numerical computing
  • scipy - Scientific computing
  • scikit-learn - Machine learning
  • matplotlib - Visualization
  • seaborn - Statistical visualization
  • jupyter - Notebooks
2 Install Database & ETL Tools
pip install sqlalchemy pymysql psycopg2-binary boto3 pyspark
  • sqlalchemy - SQL toolkit
  • pymysql - MySQL driver
  • psycopg2 - PostgreSQL driver
  • boto3 - AWS SDK
  • pyspark - Apache Spark
3 Install Development Tools
pip install pytest black flake8 mypy
  • pytest - Testing framework
  • black - Code formatter
  • flake8 - Linter
  • mypy - Type checker
4 Create requirements.txt

Save your dependencies for sharing:

pip freeze > requirements.txt

Others can install with:

pip install -r requirements.txt

Git & GitHub Setup

1 Install Git

Windows: Download from git-scm.com

macOS:

brew install git

Linux:

sudo apt install git
2 Configure Git
git config --global user.name "Your Name" git config --global user.email "your.email@example.com"
3 Create GitHub Account

Sign up at github.com

4 SSH Key Setup (Optional but Recommended)
ssh-keygen -t ed25519 -C "your.email@example.com"

Follow the prompts, then:

# Display your public key cat ~/.ssh/id_ed25519.pub

Add the key to GitHub Settings > SSH and GPG keys > New SSH key

Database Tools

1 DBeaver (Recommended)

Universal database tool - works with all databases

# Windows/macOS: Download from https://dbeaver.io/download/ # Linux: sudo apt install dbeaver-ce
2 MySQL Workbench

Specialized MySQL management tool

# Download from: https://dev.mysql.com/downloads/workbench/
3 pgAdmin (PostgreSQL)

Web-based PostgreSQL management

# Download from: https://www.pgadmin.org/download/
4 Docker Containers (Easy Setup)

Run databases in containers without installation:

# MySQL docker run --name mysql -e MYSQL_ROOT_PASSWORD=root -p 3306:3306 -d mysql:latest # PostgreSQL docker run --name postgres -e POSTGRES_PASSWORD=root -p 5432:5432 -d postgres:latest # MongoDB docker run --name mongo -p 27017:27017 -d mongo:latest

Terminal/Command Line Basics

Essential Commands
# Navigation cd path/to/directory # Change directory pwd # Print working directory ls # List files (Linux/macOS) dir # List files (Windows) # File Operations mkdir folder_name # Create directory touch file.txt # Create file rm file.txt # Delete file rm -r folder_name # Delete folder recursively # File Content cat file.txt # Display file content grep "text" file.txt # Search for text tail -n 10 file.txt # Show last 10 lines # Permissions (Linux/macOS) chmod +x script.sh # Make executable sudo command # Run with admin privileges # Useful Utilities python script.py # Run Python script pip install package # Install Python package git status # Check git status history # Show command history
Tip: Use arrow keys to navigate command history
