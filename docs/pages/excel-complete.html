Excel for Data Engineering

Excel is still one of the most widely used tools for exploring data, quick analysis, and sharing results with non‑technical stakeholders.

As a data engineer, you can use Excel to:

  • Validate small samples from your pipelines (e.g., CSV exports).
  • Create quick summary reports for business teams.
  • Prototype transformations before automating them with Python/SQL.

Formulas & Functions (Basics)

Math & Aggregation

  • SUM, AVERAGE, MIN, MAX – basic aggregations.
  • COUNT, COUNTA, COUNTIF – counting rows and conditions.
  • IF – simple conditional logic.
  • AND, OR – combine multiple conditions.
=IF(C2 >= 1000, "High Value", "Standard")

Text Functions

  • LEFT, RIGHT, MID – extract text segments.
  • LEN – length of text (useful for validation).
  • TRIM – remove extra spaces (great for cleaning).
  • CONCAT / TEXTJOIN – combine columns.
=TRIM(A2) & " " & TRIM(B2)

Lookup Functions (VLOOKUP, INDEX/MATCH, XLOOKUP)

Lookup functions let you join tables in Excel, similar to SQL joins.

VLOOKUP

=VLOOKUP(A2, Customers!$A$2:$D$1000, 3, FALSE)

Look up the customer ID in A2 and return the 3rd column from the Customers table.

INDEX + MATCH

=INDEX(Customers!$C$2:$C$1000, MATCH(A2, Customers!$A$2:$A$1000, 0))

More flexible than VLOOKUP because the key does not need to be in the first column.

Pivot Tables (Analysis)

Pivot tables give you drag‑and‑drop analytics without writing code.

  • Drag fields to Rows, Columns, and Values.
  • Use filters and slicers to explore subsets.
  • Use Value Field Settings to choose SUM, COUNT, AVERAGE, etc.
Use pivot tables to quickly compare your Excel aggregates with your SQL/warehouse results.

Data Cleaning

  • Remove duplicates with Data → Remove Duplicates.
  • Use filters/sorting to spot missing or strange values.
  • Use Text to Columns to split combined fields.
  • Use conditional formatting to highlight outliers.

Power Query

Power Query lets you connect to files, databases, and web sources and build repeatable transformation steps.

  • Connect to CSV, Excel, SQL Server, and more.
  • Apply step‑by‑step transformations: filter, split, merge, pivot/unpivot.
  • Refresh the query when new data arrives – no manual rework.
Prototype in Power Query first, then translate successful transformations into Python/SQL pipelines.

Charts & Dashboards

  • Use bar/line charts for trends and comparisons.
  • Combine pivot tables and charts into dashboards.
  • Use slicers/timelines for interactive filtering.
  • Link dashboards to CSV/Excel exports from your warehouse.
