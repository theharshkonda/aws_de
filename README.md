# AWS Data Engineer - 2 Month Intensive Bootcamp

## Overview
This comprehensive bootcamp is designed to take you from beginner to interview-ready AWS Data Engineer in 8 weeks. The curriculum covers all essential tools, technologies, and hands-on projects using AWS Free Tier.

## Program Structure

### Week 1-2: Python & SQL Fundamentals
- Python programming basics to advanced
- SQL query writing and optimization
- Data manipulation with Pandas & NumPy
- **Goal**: Build strong foundation for data engineering

### Week 3: Statistics & Data Visualization
- Descriptive and inferential statistics
- Data visualization with Matplotlib & Seaborn
- **Mini Project**: Data analysis and visualization

### Week 4: AWS Fundamentals
- AWS account setup and IAM
- S3 (Object Storage)
- RDS (Relational Database Service)
- Lambda (Serverless Computing)
- **Goal**: Master core AWS services

### Week 5: AWS Data Services
- AWS Glue (ETL)
- Amazon Athena (Query S3 data)
- Amazon Redshift (Data Warehouse)
- **Project**: Build ETL Pipeline (S3 â†’ Glue â†’ Athena)

### Week 6: Streaming & Advanced Processing
- Amazon Kinesis (Streaming)
- Amazon EMR & Apache Spark
- AWS Step Functions (Orchestration)
- **Project**: Real-time streaming pipeline

### Week 7: Business Intelligence Tools
- PowerBI
- Tableau
- AWS QuickSight
- **Project**: Create interactive dashboards

### Week 8: Capstone Project
- End-to-end data engineering project
- AWS certification preparation
- Portfolio building

All projects are designed to run within AWS Free Tier limits:

- Glue: 1M requests/month (first month)
- Athena: 5GB query data/month
- RDS: 750 hours/month
